### Сборка под проект FastAPI

- **/app** - корневая дирректория проекта FastAPI
  - **/configuration** - конфигурация приложения (регистрация путей и событий)
    - **Server.py** - модуль для генерации FastAPI приложения, регистрации путей и регистрации событий
    - **/routes** - модуль для регистрации путей
      - **\__init__.py** - `__routes__` - сбор всех путей из /internal/routes
      - **register_routes.py** - регистрация всех путей из переменной `__routes__` в `dataclass Routes`
  - **/packages** - хранение описания взаимодействия со сторонними сервисами (sqlalchemy, redis)
  - **/helpers** - модуль "функций-помощников" пример:
    - **combined_modules** - функция для сбора модулей в лист (используется для сбора маршрутов)
  - **/internal** - логика и взаимодействия внутри FastAPI приложения
    - **/routes** - определение и хранение путей приложения
    - **/events** - определение и хранение событий приложения
- **docker** - дирректория для хранения Dockerfile(ов) необходимых сервисов
- **docker-compose.yaml** - файл "оркестрации образами"
- **requirements.txt** - зависимости проекта

Установка окружения:
```
pip install -r requirements.txt
```

Запуск приложения:
```
uvicorn app:create_app --port 8000
```
